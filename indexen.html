Trainings

JavaScript and Typescript

JavaScript is an extremely rapidly evolving language. Understating its mechanics, different kind of paradigms that can be applied and modern features are vital in order to create high quality code.

Angular

Anuglar2 is a game changer in a world of web applications. Along with with RxJS and TypeScript it is a very powerful but also complex tool. Deep understanding of Angular2 mechancs allows to create a very scalable application structure and is clean, fast and easy to extend.

React + Redux

React is a powerful JavaScript library that comes along with component-based architecture and Virtual DOM. This library combined with Redux - the state managing library - provides with a very fresh approach towards creation of front-end applications.

RxJS

Reactive programming is a paradigm thanks to which dealing with asynchronous data is not longer problematic. RxJS provides with the great toolset for reactive programming, that can be used to simplify application logic or event to create entire application architecture in reactive and modular approach - Model-Viewer-Intent.

Electron and NativeScript

JavaScript is a language that is no longer limited just to web browser applications. With Electron and NativeScript it is possible to create truly end to end application that will work also on mobile devices and desktop. All with a single codebase.

GraphQL

GraphQL provides with a new approach towards fetching data from server-side that is far less limited than REST. It can be used along with any front-end framework or even with Vanilla JavaScript.

Automation

Grunt, Gulp, Webpack

Managing dependencies, reloading a page every time a change is introduced to the code... It can be really cumbersome. Thankfully, there are great tools to make the development proccess faster and easier.

Node.js + Express

JavaScript on the server-side makes it possible to have client and server code in one language. Node.js is a very powerful tool that allows to write a backend code that is fast but also can be very complex. That is why it is vital to know the best practices of writing Node.js applications

Preprocessors

LESS, Sass, SCSS and Pug (former Jade)

Making a scalable CSS and HTML structure can be very difficult. Thankfully, there are tools that help making code faster, cleaner and more joyful.

Theory

Real life examples

Exercises

A lot of practical exercises, coding and solving problems what makes it possible to work with the technology after the training.
